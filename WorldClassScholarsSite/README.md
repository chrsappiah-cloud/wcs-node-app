# World Class Scholars Landing Page

A simple landing page scaffold for World Class Scholars.

## Structure

- `index.html` — homepage content and layout
- `styles.css` — responsive styling
- `script.js` — optional interaction and onboarding behavior

## Getting started

1. Open `index.html` in a browser.
2. Customize the headline, services, contact email, and visual styling.

## Notes

- This is a static HTML/CSS/JS project scaffold.
- To deploy, host the folder on any static hosting service like GitHub Pages, Netlify, or Vercel.
